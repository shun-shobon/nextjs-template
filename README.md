# Next.js template

## Usage

### Install

```shell
$ yarn install
```

### Build

```shell
$ yarn build
```

### Start production server

```shell
$ yarn start
```

### Start development server

```shell
$ yarn dev
```

### Start storybook server

```shell
$ yarn storybook
```

### Run all linter

```shell
$ yarn lint
```

### Run all auto fix

```shell
$ yarn fix
```

### Run test

```shell
$ yarn test
```
